runbook_set

📋 描述

RunbookSet 用于定义实体、数据的操作手册集合，包含分析工具、操作指南和最佳实践。

🏷️ 版本信息

版本 v1.0.0

common_schema_info
object
背景：在阿里云可观测业务使用中，内置的UModel数据会作为公共UModel（CommonSchema）默认存在，公共UModel数据以引用方式（CommonSchemaRef）配置在UModel，在查询时动态生成UModel实例并附加该字段作为额外说明。 使用方式：该字段不需要手动配置，仅供查看。若存在该字段，则说明对应的UModel实例只可查看，不支持编辑或删除；若不存在该字段，则可根据业务需求按需编辑。
